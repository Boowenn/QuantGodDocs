# Contribution policy

## Backend PR

Use for:

- API route changes
- Python tools
- MQL5/MT5 logic
- tests and safety guards

Must update docs when API or operator workflow changes.

## Frontend PR

Use for:

- Vue components
- charts
- Ant Design UI
- service wrapper changes

Do not add backend logic or local file scraping.

## Infra PR

Use for:

- workspace scripts
- Cloudflare
- deployment/sync automation

Do not add trading logic.

## Docs PR

Use for:

- runbooks
- architecture
- API contract references
- phase status
